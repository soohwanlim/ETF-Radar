## 1. 프로젝트 개요

### 1.1 배경

국내 ETF 시장은 빠르게 성장하고 있지만, 개인 투자자가 ETF를 비교하고 분석할 수 있는 무료 도구는 부족하다. 특히 **ETF 구성종목이 바뀌었을 때 이를 감지해서 알려주는 서비스는 현재 존재하지 않는다.**

(예: TIGER 코리아원자력 ETF의 1위 구성종목이 현대건설 → 두산에너빌리티로 교체된 것을 투자자가 뒤늦게 알게 되는 문제)

### 1.2 목표

- 국내 코스피 ETF를 기간별 수익률, 테마별로 쉽게 탐색
- ETF 다중 비교 및 포트폴리오 시뮬레이션 제공
- **업계 최초** ETF 구성종목 변경 감지 및 웹 표시
- **월 운영비 $0** 저비용 아키텍처로 실제 서비스 운영
- **Antigravity 2.0**으로 개발 — 독자도 무료로 따라 만들 수 있음

### 1.3 타겟 유저

- 국내 ETF에 관심 있는 개인 투자자 (20~40대)
- 테마 투자에 관심 있는 투자자
- 구성종목 변경을 빠르게 파악하고 싶은 액티브 투자자

## 2. 핵심 기능 명세

### 2.1 기간별 수익률 랭킹 — P0

- 1주 / 1개월 / 3개월 / 1년 탭
- ETF 이름, 코드, 수익률, AUM, 운용보수, 거래량 표시
- 정렬: 수익률 높은 순 (기본) / AUM 큰 순 / 운용보수 낮은 순

### 2.2 테마별 핫 ETF — P0

- 1주 / 1개월 기준 핫한 테마 랭킹
- 테마 클릭 → 해당 테마 ETF 목록 + 구성종목 상위 10개
- 초기 테마: 반도체 / AI·로봇 / 조선·방산 / 원자력 / 2차전지 / 바이오 / 금융 / 리츠 / 미국주식 / 채권

### 2.3 구성종목 변경 감지 ⭐ — P0 (핵심 차별점)

매일 장 마감 후 KRX 구성종목 자동 비교 → 변경사항 웹 표시

| 감지 유형 | 예시 | 표시 |
|----------|------|------|
| 1위 종목 교체 | 현대건설 → 두산에너빌리티 | 🔄 빨간 배지 |
| 신규 편입 | XX종목 새로 들어옴 | 🆕 초록 배지 |
| 완전 편출 | YY종목 빠짐 | ❌ 회색 배지 |
| 비중 대폭 변화 | 15% → 32% | 📈 주황 배지 |
| Top5 순위 변동 | 3위 → 1위 | ↑↑ 표시 |

- 90일치 스냅샷 보존 → ETF 상세 페이지에서 타임라인 확인 가능

### 2.4 ETF 상세 페이지 — P0

- 기본 정보: 운용사, 상장일, AUM, 운용보수, 분배금
- 수익률 차트 (1주/1개월/3개월/1년)
- 구성종목 Top 10 + 비중 바 차트
- 섹터 비중 파이 차트
- **구성종목 변경 이력 타임라인** (최근 90일)

### 2.5 ETF 비교 — P1

- 최대 4개 ETF 선택 → 수익률, AUM, 운용보수, 구성종목 겹침 비율 나란히 비교
- 수익률 레이스 꺾은선 차트
- UX: 하단 플로팅 바에서 선택 → 비교 페이지 이동

### 2.6 포트폴리오 시뮬레이터 — P2

- ETF 여러 개 + 비중 슬라이더 → 기간별 백테스트
- 투자 권유 아님 명시 필수

### 2.7 내 ETF 알림 — P3 (Phase 3)

- Phase 2: 즐겨찾기 ETF 등록 (localStorage)
- Phase 3: 구성종목 변경 시 웹푸시/이메일 알림 (로그인 필요, Supabase)

## 3. 기술 아키텍처

### 3.1 전체 구조

```
[Antigravity 2.0]  ← 개발 환경 (Gemini 3.5 Flash 기반)
       │
       ▼ 빌드·배포
[GitHub]
       │ 자동 배포 (Push → 빌드 → 배포)
       ▼
[Cloudflare Pages]           ← React + Vite 정적 파일 호스팅
       │
       ▼ API 요청
[Cloudflare Workers]         ← 프록시 + 변경 감지 스케줄러
       │
       ├── [Workers KV]      ← 구성종목 스냅샷 (90일), 시세 캐시
       │
       ▼
[KRX 공공 데이터 API]        ← ETF 시세, 수익률, 구성종목 (무료 EOD)
```

### 3.2 Phase별 데이터 전략

| 데이터 | 소스 | 갱신 | 저장 |
|--------|------|------|------|
| ETF 기본 정보 | KRX / 수동 | 월 1회 | JSON 파일 |
| ETF 시세·수익률 | KRX API | 1일 1회 | Workers KV |
| 구성종목 현황 | KRX API | 1일 1회 | Workers KV |
| 구성종목 스냅샷 | KRX API | 1일 1회 누적 | Workers KV (90일) |
| 즐겨찾기 (P2) | - | 실시간 | localStorage |
| 유저·알림 (P3) | - | - | Supabase 무료티어 |

### 3.3 비용 구조

| 서비스 | 무료 한도 | 비용 |
|--------|----------|------|
| Cloudflare Pages | 무제한 | $0 |
| Cloudflare Workers | 10만 req/일 | $0 |
| Workers KV | 10만 read/일, 1GB | $0 |
| Workers Cron | 무료 | $0 |
| Antigravity 2.0 | 무료 (일 한도 있음) | $0 |
| Supabase (P3) | 500MB, 5만 MAU | $0 |
| Resend 이메일 (P3) | 월 3,000건 | $0 |

**월 운영비: $0** (도메인 제외, Phase 3까지)
